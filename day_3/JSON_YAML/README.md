
## JSON and YAML functions

| Name           | Input                 | Output                |
|----------------|-----------------------|-----------------------|
| json.dumps     | python dict           | json formatted string |
| json.dump      | python dict           | json file             |
| json.loads     | json formatted string | python dict           |
| json.load      | json file             | python dict           |
| yaml.dump      | python dict           | yaml file or string   | depends on if file is specified
| yaml.safe_load | yaml file or string   | python dict           |
