Intro to Git
====
____

Version Control
----

- Record changes to files over time
- Can be used to revert/compare changes
- Records who commited changes
- allows rollback
- allows multiple users to work on one file without disrupting

### Manual VC
- backup files with renaming systems
- time-consuming
- could get confusing

Early systems were more labour intensive

### Centralised VS Distributed

#### Centralised
- download file from repo
- locked if file already in use
- only one person can work on a file at a time

#### Distributed
- download a copy of entire remote repo (clone)
- can pull changes others have made to the file from remote repo
- can push your changes back to remote repo
- multiple users can work on files simultaneously
